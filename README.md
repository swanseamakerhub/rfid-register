# rifd-register
Python / QT script that allows users to sign-in to a lab using an RFID tag

# Requirements
* Python3
* PyQT5
* MFRC522 Python Library
* MariaDB

# Installation
1. Linux box with pre-reqs installed
2. Clone repo
3. Create MySQL Tables (SQL files are in repo)
4. Run python file
5. Profit!
